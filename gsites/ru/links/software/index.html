Программные инструменты для математиков

Среда для работы с графами newGRAPH

Отличный инструмент для формулирования и проверки гипотез в теории графов. Позволяет создавать графы и вычислять многие их параметры. Программа написана на Java и позволяет легко создавать подключаемые модули. К сожалению, программа не развивается в последние годы.

Среда для работы с графами Graphlab

Аналог newGRAPH, более новый по времени создания и также написанный на Java. У программы есть и своя страница на SourceForge: http://sourceforge.net/projects/graphlab/
