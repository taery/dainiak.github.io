Оптический конструктор

Программа позволяет строить оптические схемы из линз, зеркал, источников света и ширм, и наглядно демонстрировать построение изображений предметов на основе законов геометрической оптики.
Изначально программа была создана в качестве выпускной дипломной работы в Лицее Информационных Технологий №1533.
Скачать программу (100 Kb). В архиве с программой есть также подархив с исходными текстами (в формате 7-zip).
