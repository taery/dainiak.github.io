Дискретные структуры (МФТИ, 2013—2014)

Информация о курсе

Пожалуйста, ознакомьтесь с этим текстом, чтобы представлять критерии выставления оценок в весеннем семестре. Требования к теорлисткам (сроки сдачи и ограничения на объём) публикуются на отдельной странице. Все утверждения курса разбиты на уровни «ознакомительный», «стандартный», «продвинутый». Можно ознакомиться с предварительными списками вопросов для экзаменационных билетов устной части для уровней

  • ознакомительный,
  • стандартный,
  • продвинутый.
В каждом билете будет по два вопроса из этого списка.

Простые вопросы по лекциям для самоконтроля

В этом файле собраны вопросы, на которые слушатели курса должны легко уметь отвечать после каждой лекции. Если возникают сложности с ответами на эти вопросы, нужно тщательнее разобраться с материалом лекции.

Путеводитель по курсу

Здесь можно посмотреть карту основных понятий и теорем, составляющих курс. Она поможет связать эти понятия между собой и легче их запомнить. Видеокомментарий к этой карте можно посмотреть на YouTube.

Программа курса и литература

В электронном виде доступна методичка, содержащая часть материала курса.

Программа курса в этом году практически полностью повторяет программу прошлого года, но изложение ведётся в другом порядке. Ниже приводится содержание лекций со ссылками на слайды. Точный порядок, в котором читался материал, можно посмотреть на отдельной странице. Слайды лекций доступны на SlideShare.

    Элементарная комбинаторика

  • Мотивация курса. Некоторые базовые обозначения. Основные правила комбинаторики: правило сложения, правило умножения. Комбинаторные числа. Размещения, перестановки и сочетания. Числа Стирлинга второго рода и Белла.
  • Бином Ньютона, полиномиальная формула. Свойства биномиальных коэффициентов.
  • Формула включений-исключений. Неравенства Бонферрони.
  • Задачи о разбиениях чисел на слагаемые. Упорядоченные и неупорядоченные разбиения. Рекуррентные формулы. Число упорядоченных разбиений. Диаграммы Юнга. Теоремы Эйлера о равенстве количеств неупорядоченных разбиений. Формула Харди—Рамануджана (б/д). Простые рекуррентные соотношения для количества упорядоченных неупорядоченных разбиений.

    • Некоторые классические вопросы комбинаторики

  • Основные понятия асимптотического анализа. Символы Ο, Ω, Θ, ∼, ≲. Примеры. Оценки для факториалов. Формула Стирлинга (б/д). Оценки биномиальных коэффициентов.
  • Теорема об асимптотической оценке функций, удовлетворяющих рекуррентному неравенству (пример применения: сортировка слиянием).
  • Линейные рекуррентные соотношения с постоянными коэффициентами. Общий вид решения: случай различных корней характеристического многочлена — с доказательством, общий случай — только формулировка. Пример применения: подсчёт количества слов с запретами, числа Фибоначчи.

    • Элементарная теория графов

  • Основные определения теории графов. Маршруты, расстояния в графах. Двойной подсчёт, теорема «о рукопожатиях».
  • Деревья. Шесть эквивалентных определений деревьев. Утверждение об одной/двух центральных вершинах в дереве.
  • Раскраски графов. Хроматическое число и хроматический индекс, их простые верхние и нижние оценки. Теоремы Брукса и Визинга (без доказательства). Жадный алгоритм раскраски.
  • Планарные графы. Формула Эйлера. Оценка числа рёбер в планарном графе. Миноры. Критерий планарности Вагнера. Хроматическое число планарных графов. Теорема о пяти красках.
  • Теорема Холла о паросочетаниях в двудольных графах. Следствие о хроматическом индексе двудольных графов.
  • Эйлеровы и гамильтоновы циклы в графах. Использование эйлеровых циклов для построения последовательностей Де Брёйна.
  • Изоморфизм графов. Автоморфизмы.

    • Некоторые классические вопросы теории графов

  • Теорема Турана. Теорема Эрдёша—Стоуна (б/д).
  • Проблема Заранкевича: формулировка на языке двудольных графов и матриц. Числа Заранкевича. Асимптотика чисел Заранкевича для специальной последовательности параметров.
  • Теорема Рамсея (для графов для случая двух цветов). Числа Рамсея.

    • Алгебраические структуры

  • Группы: определение, примеры. Изоморфизм групп. Понятие подгруппы.
  • Задача подсчёта количества раскрасок комбинаторной конфигурации. Лемма Бёрнсайда. Теорема Редфилда—Пойи.
  • Группы вычетов: аддитивные и мультипликативные. Порядок элемента. Циклические группы. Теорема Эйлера—Ферма.
  • Поля. Неприводимые многочлены: их свойства и количество. Построение полей на основе неприводимых многочленов.
  • Частично упорядоченные множества. Булеан. Минимаксные теоремы о разложении ч.у.м. на цепи и антицепи.

    • Гиперграфы и системы множеств

  • Гиперграфы. Жадный алгоритм построения покрытия гиперграфа/матрицы. Оценка мощности «жадного» покрытия, конструкция матрицы с большой глубиной.
  • Теоремы о пересечении конечных множеств: теоремы Фишера, Эрдёша—Ко—Радо, Альсведе—Хачатряна (без доказательства), Ловаса (гипотеза Кнезера).
  • Размерность Вапника—Червоненкиса. Верхняя оценка мощности семейства через VC-размерность. Оценка размерности измельчения. ε-сети. Теорема Хаусслера—Вельцля.

    • Продвинутые методы комбинаторики

  • Степенные ряды и производящие функции. Пример тождества, доказываемого с помощью степенных рядов. Производящие функции последовательностей, удовлетворяющих линейным рекуррентным соотношениям. Числа Каталана. Нахождение сумм с участием биномиальных коэффициентов, чисел Фибоначчи и т.д.
  • Продвинутые применения метода производящих функций и степенных рядов: рекуррентное соотношение для количества неупорядоченных разбиений (с участием пентагональных чисел), количество неприводимых нормированных многочленов над Zp.
  • Обращение Мёбиуса на частично упорядоченных множествах. Формула включения-исключения и арифметическая формула обращения как частные случаи общей теории обращения Мёбиуса. Применение арифметической формулы обращения Мёбиуса для подсчёта числа циклических слов.
  • Nullstellensatz-метод. Теорема о существовании необнуляющего набора. Примеры применения метода: теорема Коши—Давенпорта, теорема Алона—Фюреди о покрытии гиперкуба гиперплоскостями, теорема Алона—Фридланда—Калаи о регулярных подграфах в почти регулярных графах.
  • Вероятностный метод в комбинаторике. Простая вероятностная нижняя оценка чисел Рамсея.
  • Применения вероятностного метода: существование графов с большим обхватом и хроматическим числом, оценка числа скрещиваний, нижняя оценка числа независимости через степени вершин, нижняя оценка чисел Заранкевича.
  • Локальная лемма Ловаса. Применение к нижней оценке диагональных и внедиагональных чисел Рамсея.
  • Линейно-алгебраический (ранговый) метод. Применение для явного построения рамсеевских графов.
