Дискретная оптимизация (МФТИ, весна 2014)

Эволюционировавшая версия курса находится здесь.

Краткое содержание

  1. Примеры задач дискретной оптимизации. Отличия от задач непрерывной оптимизации: примеры возникающих трудностей.
  2. Задача линейного программирования. Пример задания задач дискретной оптимизации в терминах ЛП: условия Миллера—Таккера—Землина.
  3. Локальный поиск в дискретной оптимизации. Окрестностные функции. Пример: 2-окрестности гамильтоновых циклов.
  4. Дискретная линейная задача о подмножествах. Формулировки задач TSP и MST. Эквивалентность поиска минимального и максимального остовного дерева. Жадный алгоритм. Матроиды и теорема Радо—Эдмондса.
  5. Лемма об изолировании: вероятность единственности решения дискретной линейной задачи о подмножествах.
  6. Задачи о покрытии. Матричное представление, версия с весами. Анализ качества решения, получаемого жадным алгоритмом.
  7. Оптимизация минимумов, сумм и произведений. Теоремы Гросса и Гермейера.
  8. Решение задач ЛП. Симплекс-метод: локальный поиск по вершинам многогранника. Алгебраическая формулировка. Организация симплекс-таблицы.
  9. Двойственность в линейном программировании: существование сертификата оптимальности решения. Простой приближённый алгоритм для задачи о взвешенном вершинном покрытии: решить задачу ЛП, округлить. «Комбинаторный» алгоритм, концептуально основанный на двойственности.
  10. Задачи оптимизации на графах. MST, TSP, паросочетания. Паросочетание в двудольных и произвольных графах. Алгоритм Эдмондса.
  11. Комбинаторный перебор. Универсальные стратегии неизбыточного перебора дискретных объектов. Некоторые методы перебора подмножеств, перестановок, сочетаний, и разбиений.

  • Friedrich Eisenbrand's course Linear and Discrete Optimization on Coursera.
  • Pascal Van Hentenryck's course Discrete Optimization on Coursera.
  • Ant Colony Optimization and Evolutionary Multiobjective Optimization demos on Wolfram Demonstrations Project.
  • Graphical Linear Programming for Two Variables, Graphical Linear Programming for Three Variables, The Fundamental Theorem of Linear Programming and Graph of Inequalities demos on Wolfram Demonstrations Project.
  • Бесплатный пакет для решения задач линейной оптимизации LPSolve. Для многих языков программирования существуют библиотеки, позволяющие пользоваться функциями LPSolve. Пример скрипта для LPSolve.
  • Мощная программная среда для решения оптимизационных задач IBM ILOG CPLEX с собственным языком программирования OPL. Можно бесплатно скачать и пользоваться в течение 90 дней.
