Дискретные структуры (МФТИ, 2011—2012)

Эволюционировавшую версию курса можно найти на этой странице.

Программа курса

  • Основные правила комбинаторики: правило сложения, правило умножения, принцип Дирихле, формула включений и исключений. [ВВВ, §§4,13,131] [Ша, гл. 1]
  • Размещения, перестановки и сочетания. Бином Ньютона, полиномиальная формула. Некоторые тождества с биномиальными коэффициентами. [ВВВ, §§5,19,20,23,24,33] [Ша, гл. 1] [3, гл. 8]
  • Оценки для факториалов и биномиальных коэффициентов. Формула Стирлинга (б/д). Запись (c+o(1))n. Оценки биномиальных коэффициентов. Аналогичные результаты для полиномиальных коэффициентов. Оценки для C_n^k при бо́льших k. [ГКП, §4.4 и гл. 9] [КЛРШ, гл. 3]
  • Частично упорядоченные множества. Разложение ч.у.м. на цепи и на антицепи. Вывод теоремы Холла о системах различных представителей. [Ju, §9.1] [Ан, §2.6]
  • Функция Мёбиуса на ч.у.м.. Формула обращения Мёбиуса. «Арифметическая» формула обращения Мёбиуса и формула включения-исключения как частные случаи. Применение формулы обращения Мёбиуса для подсчёта числа циклических последовательностей. [Са, гл. 2 §3]
  • Задачи о разбиениях чисел на слагаемые. Упорядоченные и неупорядоченные разбиения. Рекуррентные формулы. Число упорядоченных разбиений. Диаграммы Юнга. Теоремы Эйлера о равенстве количеств неупорядоченных разбиений. Формула Харди—Рамануджана (б/д). [ВВВ, гл. 4]
  • Линейные рекуррентные соотношения с постоянными коэффициентами. Соотношения 2-го порядка — с доказательством, соотношения большего порядка — только формулировка. Числа Фибоначчи. [ВВВ, §§94, 101—104] Степенные ряды и производящие функции. Пример тождества, доказываемого с помощью степенных рядов. Теоремы о сходимости степенных рядов (б/д). Числа Каталана. Нахождение сумм с участием биномиальных коэффициентов, чисел Фибоначчи и т.д. [ВВВ, гл. 8] [ГКП, гл. 7] [Ан, гл. 8]
  • Алгебраические структуры: группы, кольца, поля. Аксиоматика. Примеры конечных и бесконечных групп, колец и полей. [Любой учебник общей алгебры] [КМ, §1] [ББ, §7.3, 7.5, 10.1] [Ан, §§9.4,20.1] [Со, гл. 6]
  • Теорема Кэли об универсальности групп подстановок. Понятия действия группы на множестве и орбиты элемента. Примеры. [КМ, §11, 12]
  • Разложение группы по подгруппе. Фактор-группа по нормальной подгруппе. Теорема Лагранжа. Теорема Силова. [КМ, §§2, 11.1] [ББ, §§7.9—7.11]
  • Лемма Бернсайда. Перечислительная теорема Пойи. [ВВВ, гл. 9] [Ан, гл. 19]
  • Элементы теории кодирования. Неравенство Макмиллана. Теорема о существовании префиксных кодов с заданными длинами кодовых слов. Кодовое расстояние и его связь с устойчивостью к ошибкам. Линейные коды. Коды Хемминга. [Ан, гл. 18] [Со, гл. 1]
  • Теоремы о нулях полиномов. Применение: теорема о существовании в заданном графе регулярного подграфа. [Al]
  • Матрицы Адамара. Нормализация. Необходимое условие существования. Гипотеза Адамара. Достаточные условия существования. Коды на основе матриц Адамара. [Со, §9.1]
  • Определение графа, орграфа, мультиграфа, псевдографа и т.д. Понятие гиперграфа. Изоморфизм графов. Число независимости, кликовое число, хроматическое число и соотношения между ними. [ЕМСТ, §§1,2,4,25,26,53]
  • Теория Рамсея. Числа Рамсея R(s,t): точные значения для s=1,2; верхняя оценка Эрдёша—Секереша s+ts, её следствие для диагональных чисел Рамсея; нижняя оценка диагональных чисел с помощью простого вероятностного метода. Теорема Рамсея для гиперграфов (б/д). Теорема Рамсея для бесконечных множеств. [АС, §3.1]
  • Лемма Ловаса: симметричный и общий случаи. Идея уточнения нижней оценки чисел Рамсея с её помощью. [АС, §§5.1,5.3]
  • Линейно-алгебраический метод: теорема Франкла—Уилсона, конструктивная нижняя оценка чисел Рамсея R(s,t). [РаЛ, гл. 5] [РаЭ, гл. 5]
  • Маршруты в графах. Гамильтоновы и эйлеровы циклы. Достаточное условие существования гамильтонова цикла (б/д). Критерий существования эйлерова цикла. Построение последовательностей Де Брёйна на основе эйлеровых циклов. [ЕМСТ, §§43,44] [РаЭ, гл. 9]
  • Графы пересечений и непересечений (кнезеровские графы). Теорема Эрдёша—Ко—Радо. [РаЭ, §10.1] Кнезеровский граф (граф непересечений для полного однородного гиперграфа). Верхняя оценка его хроматического числа. Простые нижние оценки. Примеры конкретных кнезеровских графов. Теорема Борсука—Улама—Люстерника—Шнирельмана (б/д). Теорема Ловаса о хроматическом числе кнезеровского графа. (Можно заменить комбинаторным доказательством Матушека.) [РаЭ, §10.1]
  • Теорема Турана о числе рёбер в графе c заданными количеством вершин и числом независимости. [АЦ, разд. 32]
  • Системы общих представителей (с.о.п.). Переформулировка в терминах покрытия матриц. Величина (n,s,k). «Тривиальные» нижние и верхние оценки. Верхняя оценка с помощью жадного алгоритма. Конструктивная нижняя оценка. Вероятностная нижняя оценка. Связь верхней и нижней оценок. [РаС, гл. 1—4]
  • С.о.п. в геометрии (теорема о треугольниках на плоскости). Размерность Вапника—Червоненкиса. Подсчет размерности пространства (Rn,H): теорема Радона (б/д). Лемма о числе областей в пространстве заданной мощности и размерности. Лемма о размерности измельчения. [РаС, гл. 6]
  • Сертификаты («свидетели») двоичных векторов. Теорема Бонди. Верхняя оценка среднего размера сертификата. [Ju, §12.1,12.2]
