Дискретные структуры (МФТИ, 2014—2015)

Важная информация

  • Описание того, как будет проходить экзамен
  • Вопросы, из которых будут формироваться экзаменационные билеты устной части

Путеводитель по курсу

Здесь можно посмотреть карту основных понятий и теорем, составляющих курс. Она поможет связать эти понятия между собой и легче их запомнить. Видеокомментарий к этой карте можно посмотреть на YouTube.

Задачи

  • Листки к семинарам

Программа курса и литература

  • Методичка «Алгебраические структуры и их приложения к задачам дискретной математики», содержащая часть материала курса.
  • Страница курса прошлого года содержит слайды лекций и другие материалы.
  • Слайды лекций текущего года.
  • Содержание прочитанных лекций: актуальная информация о том, что успели пройти.

    Элементарная комбинаторика

  • Основные правила комбинаторики: правило сложения, правило умножения. Комбинаторные числа. Размещения, перестановки и сочетания. Числа Стирлинга второго рода и Белла.
  • Бином Ньютона, полиномиальная формула. Свойства биномиальных коэффициентов.
  • Формула включений-исключений. Неравенства Бонферрони.
  • Задачи о разбиениях чисел на слагаемые. Упорядоченные и неупорядоченные разбиения. Рекуррентные формулы. Число упорядоченных разбиений. Диаграммы Юнга. Теоремы Эйлера о равенстве количеств неупорядоченных разбиений. Формула Харди—Рамануджана (б/д). Простые рекуррентные соотношения для количества упорядоченных неупорядоченных разбиений. Рекуррентное соотношение для количества неупорядоченных разбиений: вывод с помощью метода производящих функций.

    • Некоторые классические вопросы комбинаторики

  • Основные понятия асимптотического анализа. Символы Ο, Ω, Θ, ∼, ≲. Примеры. Оценки для факториалов: формула Стирлинга (б/д) и несколько более грубые верхняя и нижняя оценки. Оценки биномиальных коэффициентов.
  • Линейные рекуррентные соотношения с постоянными коэффициентами. Общий вид решения: случай различных корней характеристического многочлена — с доказательством, общий случай — только формулировка. Пример применения: подсчёт количества слов с запретами, числа Фибоначчи. Производящие функции последовательностей, удовлетворяющих линейным рекуррентным соотношениям.

    • Теория графов

  • Основные определения теории графов. Маршруты, расстояния в графах. Двойной подсчёт, теорема «о рукопожатиях».
  • Деревья. Шесть эквивалентных определений деревьев. Утверждение об одной/двух центральных вершинах в дереве.
  • Формула Кэли. Кодирование Прюфера.
  • Раскраски графов. Хроматическое число и хроматический индекс, их простые верхние и нижние оценки. Теоремы Брукса и Визинга (без доказательства). Жадный алгоритм раскраски.
  • Хроматический многочлен: определение, рекуррентное соотношение, свойства некоторых коэффициентов.
  • Планарные графы. Формула Эйлера. Оценка числа рёбер в планарном графе. Миноры. Критерий планарности Вагнера. Хроматическое число планарных графов. Теорема о пяти красках.
  • Теорема Холла о паросочетаниях в двудольных графах. Следствие о хроматическом индексе двудольных графов.
  • Эйлеровы и гамильтоновы циклы в графах. Использование эйлеровых циклов для построения последовательностей де Брёйна.
  • Изоморфизм графов. Автоморфизмы.
  • Теорема Рамсея (для графов для случая двух цветов). Числа Рамсея.
  • Теорема Турана.

    • Алгебраические структуры

  • Перестановки: порядок перестановки, разложение на циклы, представление в виде произведения транспозиций.
  • Группы: определение, примеры. Изоморфизм групп. Понятие подгруппы. Теоремы Лагранжа и Силова.
  • Задача подсчёта количества раскрасок комбинаторной конфигурации. Лемма Бёрнсайда. Теорема Редфилда—Пойи.
  • Группы вычетов: аддитивные и мультипликативные. Порядок элемента. Малая теорема Ферма. Первообразные корни. Критерий существования первообразного корня по модулю целого числа.
  • Поля. Неприводимые многочлены: их свойства и количество. Построение полей на основе неприводимых многочленов.
  • Частично упорядоченные множества. Булеан. Теоремы Л—Я—М и Шпернера. Минимаксные теоремы о разложении ч.у.м. на цепи и антицепи. Вывод теоремы Холла из теоремы Дилуорта.

    • Продвинутые методы комбинаторики

  • Степенные ряды и производящие функции. Пример тождества, доказываемого с помощью степенных рядов. Числа Каталана: определение, рекуррентное соотношение, вывод явной формулы. Нахождение сумм с участием биномиальных коэффициентов, чисел Фибоначчи и т.д.
  • Обращение Мёбиуса на частично упорядоченных множествах. Формула включений-исключений и арифметическая формула обращения как частные случаи общей теории обращения Мёбиуса. Применение арифметической формулы обращения Мёбиуса для подсчёта числа циклических слов.
  • Вероятностный метод в комбинаторике. Простая вероятностная нижняя оценка чисел Рамсея.
  • Применения вероятностного метода: существование графов с большим обхватом и хроматическим числом, оценка числа скрещиваний, нижняя оценка чисел Заранкевича.
  • Локальная лемма Ловаса. Применение к нижней оценке диагональных чисел Рамсея.
  • Линейно-алгебраический (ранговый) метод. Применение для явного построения рамсеевских графов (теорема Франкла—Уилсона).
  • Алгебраические оценки чисел Заранкевича Z2(m).
