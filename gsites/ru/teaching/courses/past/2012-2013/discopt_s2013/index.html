Дискретная оптимизация (МФТИ, весна 2013)

Содержание лекций со ссылками на слайды

  1. Оргвопросы. Некоторые задачи дискретной оптимизации. Локальный поиск: пример в задаче коммивояжёра. Понятие о целочисленном линейном программировании. Сведение задачи коммивояжёра к задаче булева линейного программирования: условия Таккера.
  2. Задачи оптимизации типа, в которых оптимизируемая характеристика складывается из отдельных компонент объекта. Примеры: задачи TSP и MST. Переход в задачах TSP и MST от минимизации к максимизации. Жадный алгоритм. Матроиды, теорема Радо—Эдмондса. Лемма об изолировании: вероятность единственности решения задачи оптимизации при случайном выборе весов.
  3. Задача о покрытии. Жадный алгоритм её решения. Теоремы о связи мощностей жадного и оптимального покрытий.
  4. Некоторые метаэвристики: метод ветвей и границ, моделирование отжига, эволюционные алгоритмы, алгоритмы муравьиных колоний.
  5. Оптимизация минимумов, сумм и произведений. Постановка задачи линейного программирования.
  6. Симплекс-метод решения задач линейного программирования. Особенность задач целочисленного программирования. Вполне унимодулярные матрицы. Унимодулярность матрицы смежности в задаче о назначениях.
  7. Задача исчерпывающего перебора дискретных неизоморфных объектов. Алгоритмы упорядоченного перебора (orderly algorithm). Разбор на примере перебора графов и деревьев.
  8. Задачи исчерпывающего перебора сложных дискретных объектов. Метод обращения локального поиска. (См. статью D. Avis и K. Fukuda из списка литературы.)

Домашние задания

  • Задание по первой лекции. Скачайте и установите любой из дистрибутивов LaTeX, позволяющий работать с кириллицей и компилировать pdf-файлы. Например, MiKTeX. Научитесь набирать простейшие тексты и компилировать их в формат pdf (на это потребуется около двух часов, пригодятся эти знания потом, наверняка, не один раз). Среди прочего можно руководствоваться свободно доступной книгой С. М. Львовского Набор и вёрстка в пакете LaTeX. Скомпилируйте в LaTeX это задание и вышлите мне на почту единственный pdf-файл с решением.
  • Задание по второй лекции.
  • Задание по третьей лекции. Если Вы решаете задачу по программированию, оформите её в соответствии с требованиями.
  • Задание по четвёртой лекции.
  • Задание по пятой лекции.
  • Первое задание по седьмой лекции.
  • Второе задание по седьмой лекции.

Программа курса

  1. Примеры задач дискретной оптимизации в теории графов, теории расписаний, геометрии.
  2. Отличительные особенности задач целочисленной оптимизации. Целочисленное линейное программирование.
  3. Методы решения задач ЦЛП. Отсечение Гомори. Метод ветвей и границ.
  4. Матроиды. Задачи оптимизации, эффективно решаемые жадными алгоритмами.
  5. Жадные алгоритмы в задаче о покрытии.
  6. Классические оптимизационные задачи на графах. Подходы к решению задач дискретной оптимизации на примере задачи коммивояжёра.
  7. Метаэвристики решения оптимизационных задач.
  8. Комбинаторный перебор. Универсальные стратегии неизбыточного перебора дискретных объектов. Некоторые методы перебора подмножеств, перестановок, сочетаний, и разбиений.
  9. Задача проверки КНФ на выполнимость. SAT-решатели. Примеры сведения задач оптимизации к задаче выполнимости. Прикладные программные пакеты для решения задач дискретной оптимизации.

Дополнительные ресурсы

  • Курс Фридриха Айзенбранда Linear and Discrete Optimization на Coursera.
  • Демонстрации Ant Colony Optimization и Evolutionary Multiobjective Optimization на портале Wolfram Demonstrations Project.
  • Демонстрации Graphical Linear Programming for Two Variables, Graphical Linear Programming for Three Variables, The Fundamental Theorem of Linear Programming и Graph of Inequalities на портале Wolfram Demonstrations Project.
  • Бесплатный пакет для решения задач линейной оптимизации LPSolve. Для многих языков программирования существуют библиотеки, позволяющие пользоваться функциями LPSolve. Пример скрипта для LPSolve.
  • Мощная программная среда для решения оптимизационных задач IBM ILOG CPLEX с собственным языком программирования OPL. Можно бесплатно скачать и пользоваться в течение 90 дней.
